In this repository we provide all the needed data for the figures in our paper Linear analysis of I-C-Love universal relations for neutron stars

for data in files with name "...pRMIL...", the data contain
central pressure p_0        Radius R        Mass M        Moment of inertia I        tidal deformability \Lambda

for data in files with name "..._pert_...", the data contain
pressure        \delta R/R        \delta M/M        \delta I/I        \delta \Lambda/\Lambda

All the data are in the geometric units
